# SolarSystem

A simulation of a Solar System using Velvet-Velocity algorithm in Ruby

## Usage

```sh
ruby main.rb [planets_amount] [dt] [final_time]
```

Output in .xyz format to be used in [OVITO](http://www.ovito.org/)
